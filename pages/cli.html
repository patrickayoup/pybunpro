Command Line Interface¶

Although mainly a REST client library, PyBunpro also includes a CLI application which can be used to query the API.

pybunpro¶

pybunpro [OPTIONS] COMMAND [ARGS]...

Options

--api-key <api_key>¶

The Bunpro API key to use [required]

--debug¶

Run in debug mode

recent-items¶

pybunpro recent-items [OPTIONS]

Options

--limit <limit>¶

The max number of items to return [1-50]

study-queue¶

pybunpro study-queue [OPTIONS]
